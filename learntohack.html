Learn To Hack

A short guide! (For legal reasons, not really.)

A successful hack, in most modern systems, is gaining a remote shell access with Sudo. A remote shell allows the hacker to execute commands, and Sudo allows them to act as an administrator. The easiest way to do this is open a port on the server and connect to it.

People can hack for information, for money, for services... the motives are infinite!

Nmap (Network Mapper)
In summary, Nmap is a network security mapper (or open-source network scanner) that can find services and hosts on a network. It has several features, such as vulnerability detection and adapting to network conditions (like latency for example). It started out as a Linux (in short, a family of open-source operating systems) utility.

Nessus
Arguably one of the most famous ones, Nessus is a vulnerability scanner designed by Tenable, a cybersecurity company. It can find bugs, and detect many flaws such as “weak” passwords, and unpatched services.

Nikto
Nikto is a web server that can identify many flaws in other websites, such as outdated softwares and dangerous files. It can identify over 6400 potentially harmful CGIs (and files), check plug-inns, and find insecurities.
